# 异常效果模块 · 变量与占位符手册

## 1. 简述

本文档介绍提供的全部可用变量、占位符及其解析规则。阅读完毕后，您无需查阅任何源代码即可正确书写公式、占位符并排查常见问题。

---

## 2. 解析流程速览
1. **PlaceholderAPI 占位符** `%...%`  
   遇到 `%` 包裹的内容时，先递归解析（可无限嵌套，直至文本不再变化）。  
2. **MythicMobs 运行时变量** `{VAR:变量名}`  
3. **MythicMobs Stat 属性** `{STAT:属性名}`  
4. **原版实体属性** `{ATB:属性键}`  
5. **外部动态变量** ：
    - {deliver}
6. 完整替换后，把最终字符串送入公式计算器求值。  
7. 若解析失败或变量不存在，自动替换为 `0`，同时写入调试日志（不会抛出异常）。

---

## 3. 变量与占位符详解

### 3.1 PlaceholderAPI 占位符 `%...%`
• 写法与官方一致，可与本插件的 `ae` 扩展混用。  
• 解析时若无玩家上下文，将原样替换为 `0`，避免报错。

### 3.2 MythicMobs 变量 `{VAR:变量名}`
• 等效于 MythicMobs 内部的 `caster.var.变量名`。  
• 仅当实体为 MythicMob 且变量已存在时才返回具体数值，否则为 `0`。

### 3.3 MythicMobs Stat 属性 `{STAT:属性名}`
• 支持在 **MythicMobs属性** 中登记的任意属性，大小写与下划线不敏感。  
• 若属性不存在，返回 `0` 并在调试日志中标记未命中。

### 3.4 原版实体属性 `{ATB:...}`
以下键已内置，均不区分大小写：  

| 键名                | 对应值说明                                   |
|---------------------|----------------------------------------------|
| `DAMAGE` / `ATTACK_DAMAGE` | 实体当前攻击力（`GENERIC_ATTACK_DAMAGE`） |
| `LEVEL`             | 玩家等级；非玩家恒为 `0`                      |
| `HP` / `THP`        | 当前生命值（整数 / 原始值）                   |
| `MHP`               | 最大生命值（整数）                            |
| `PHP`               | 当前血量百分比 (0~1)                          |
| `L_X` `L_Y` `L_Z`   | 方块坐标 (整数)                               |
| `L_X_DOUBLE` `L_Y_DOUBLE` `L_Z_DOUBLE` | 精确坐标 (小数)            |
| `L_YAW` `L_PITCH`   | 朝向角度 (四舍五入整数)                       |
| `L_X_N` `L_Y_N` `L_Z_N` | 在坐标基础上随机 ±N 方块（示例：`{ATB:L_X_3}`） |

## 4. 异常效果占位符（PlaceholderAPI 标识 `ae`）

### 4.1 基本格式
1. 针对**当前玩家 / 实体**  
   ```
   %ae_<action>_<effectId>%
   ```  
2. 针对**指定 UUID 实体**  
   ```
   %ae_<action>_<effectId>_<uuid>%
   ```  
3. 施加效果（可选基础值）  
   ```
   %ae_applyeffect_<effectId>_<uuid>_<value>%
   ```
   • `<value>` 省略时默认为 `1.0`。  
   • `effectId` 支持包含下划线（如 `ice_stun`）。

### 4.2 Action 列表

| Action                | 返回类型 | 说明 |
|-----------------------|----------|------|
| `applyeffect`         | bool     | 向目标实体施加效果（返回 1=成功 / 0=失败） |
| `has`                 | bool     | 是否拥有该异常效果（积累效果按缓存存在判定；非积累效果按当前是否仍有效判定） |
| `display`             | string   | 效果显示名 |
| `type`                | string   | 效果分类 (`positive` / `negative` 等) |
| `tenacity`            | bool     | 是否启用韧性系统 |
| `tenacityreduction`   | double   | 当前韧性减免百分比 (0~1) |
| `tenacitymax`         | double   | 配置中允许的最大减免 |
| `value`               | double   | 累积值（计数型效果） |
| `currentthreshold`    | double   | 本轮触发阈值 |
| `basethreshold`       | double   | 默认初始阈值 |
| `percent`             | double   | 累积进度百分比 (0~100) |
| `stage`               | int      | 最近一次触发的阶段编号 |
| `resistance`          | double   | 每次触发后阈值递增量 |
| `decay`               | bool     | 是否开启衰减 |
| `decayrate`           | double   | 每周期衰减值 |
| `lastactivity`        | long     | 最近一次触发/累积时间戳(ms) |
| `timesince`           | double   | 距上次活动的秒数 |
| `basevalue`           | double   | 最近一次施加的基础值 |
| `instant`             | bool     | 上次是否为即时生效 |
| `timeleft`            | double   | 非累积效果剩余时间 (秒) |
| `active`              | bool     | 非累积效果是否仍有效 |
| `finalvalue`          | double   | 最近一次实际应用值 |
| `scalefactor`         | double   | 最近一次缩放系数 |

### 4.3 返回值说明

- 布尔类占位符统一返回字符串 `1`（真）/`0`（假），不是 `true`/`false`。
- 当前玩家模式若无玩家上下文，返回值由 PlaceholderAPI 上游按空值流程处理。
